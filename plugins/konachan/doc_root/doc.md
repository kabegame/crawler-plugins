# Konachan 动漫壁纸 - 插件说明

本插件用于从 `konachan.net` 网站爬取动漫壁纸并加入下载队列。

## 配置项

- **起始页面（start_page）**：要爬取的起始页面（最小值为 1）
- **结束页面（end_page）**：要爬取的结束页面。**请保持最多爬取 100 页，降低并发量文明爬取**，否则会拒绝爬取
- **质量（quality）**：
  - **高（high）**：优先下载高分辨率图片，如果不存在则降级为中等质量
  - **中（medium）**：下载中等质量图片（默认质量）

## 使用说明

1. 选择要爬取的页面范围（起始页面到结束页面）
2. 选择图片质量偏好
3. 点击"开始爬取"按钮
4. 插件会自动访问列表页，然后进入每个详情页下载图片

## 注意事项

- **请文明爬取**：一次最多只能爬取 100 页，超过限制会拒绝执行
- **结束页面必须大于等于起始页面**，否则会报错
- 如果选择"高"质量但图片不存在高分辨率版本，会自动降级为中等质量
- 建议合理设置页面范围，避免对服务器造成过大压力
