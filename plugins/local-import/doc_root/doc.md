# 本地导入（local-import）— 使用说明

该插件用于导入本地图片到 Kabegame 图库，支持两种输入方式：

- **导入单文件**：设置 `path` 为图片文件路径（也可选择 `.zip` 压缩包；会先解压到临时目录再递归导入图片，临时目录会自动清理）。
- **导入文件夹**：设置 `path` 为文件夹路径，可选 `recursive=true` 递归导入子文件夹。

## 变量说明

- `path`：本地路径（图片文件 / zip / 文件夹；自动识别）。
- `recursive`：是否递归扫描子文件夹（仅文件夹导入时生效）。
- `file_extensions`：要导入的图片扩展名列表（不包含点号，仅文件夹导入时生效）。
