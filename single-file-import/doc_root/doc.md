# 单文件导入（single-file-import）— 使用说明

本插件用于把**本地单个图片文件**导入到 Kabegame 图库（只入队这一张）。

## 适用场景

- 你只想快速把某一张图片丢进图库
- 你不想扫描整个文件夹

## 一步步怎么用

1. 打开 **画廊 / 开始导入**
2. 在“选择源”里选 **单文件导入**
3. 在“插件配置”里点击 **选择文件**，挑选一张图片
4. 点 **开始收集**

## 配置项说明

- **图片文件（file_path）**
  - 选择一个本地图片文件路径（仅导入这一张）

## 支持的文件格式

通常支持：`jpg/jpeg/png/gif/webp/bmp`（与应用内图片处理能力一致）。

## 路径格式（手动输入时）

建议使用按钮选择文件；如果你要手动输入，下面两种都可以：

- Windows 路径：`D:\Pictures\1.jpg`
- file URL：`file:///D:/Pictures/1.jpg`

## 导入规则/常见问题

- **只会入队一次吗？**
  - 是。脚本只调用一次 `download_image(file_path)`。
- **会不会把原文件删掉？**
  - 不会。后端会复制/保存到图库目录，不会删除源文件。
- **重复导入会怎样？**
  - 如果数据库里已经存在同路径的图片，后端会跳过再次入队（并在需要时补全缩略图）。
